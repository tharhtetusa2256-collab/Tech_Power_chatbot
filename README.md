# ⚡ Tech Power AI Business Assistant

A Streamlit-based AI assistant prototype for **Tech Power Co.,Ltd**. This project is designed to become the first public flagship demo for AI automation services: customer support, lead capture, content ideas, and workflow consulting for small businesses.

## 🎯 Project Goal

Tech Power Co.,Ltd is building practical AI automation systems for local and global businesses. This repository is the starting point for a reusable chatbot/assistant template that can be customized for clients such as schools, clinics, real estate teams, online shops, agencies, and service businesses.

## ✅ What This App Can Do

- Answer business and customer-support questions
- Help collect lead details from potential clients
- Suggest automation workflows for small businesses
- Generate content ideas for marketing and social media
- Demonstrate Tech Power's AI service capability to clients

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/tharhtetusa2256-collab/Tech_Power_chatbot.git
cd Tech_Power_chatbot
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the Streamlit app

```bash
streamlit run streamlit_app.py
```

4. Add your OpenAI API key in the app sidebar.

For deployed versions, store the API key in Streamlit secrets instead of hardcoding it.
